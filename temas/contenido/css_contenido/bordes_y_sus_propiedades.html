Bordes y sus propiedades.


Los bordes son propiedades de CSS que nos permiten añadir bordes o marcos a nuestras imágenes, texto o cualquier tipo de archivo, estas propiedades son muy personalizables y nos permiten añadir mas efectos visuales a nuestra pagina web. Las principales propiedades de un borde en CSS son las siguientes.

border-width. Esta propiedad como su nombre lo indica es para el ancho del borde.

border-style. Esta propiedad indica el estilo que tendra nuestro borde. Sus propiedades son:

 1. none  2. hidden  3. dotted  4. dashed  5. solid  6. double  7. groove  8. ridge  9. inset  10. outset

border-color. Esta propiedad indica el color que va a tener nuestro borde.
