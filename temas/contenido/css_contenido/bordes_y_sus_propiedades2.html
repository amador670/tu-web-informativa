Bordes y sus propiedades parte II.


En el tema anterior se explico las propiedades de un borde, como se visualiza y una breve definición. Ahora explicaremos otras propiedades de un borde, como configurar solo un determinado lado del borde (superior, derecho, inferior e izquierdo) entre otras propiedades. A continuación cada una de ellas.

Cambiar el ancho de cada lado de un borde.


border-top-width. Ancho del borde superior.
border-right-width. Ancho del borde derecho.
border-bottom-width. Ancho del borde inferior.
border-left-width. Ancho del borde izquierdo.

Con estas propiedades podemos cambiar el ancho de cada lado de nuestro borde a nuestro gusto.

Si queremos cambiar el estilo de cada lado de nuestro borde cambiamos la palabra width por style y podemos modificar el estilo de cada lado de nuestro borde a nuestro gusto.

Propiedades para cambiar el estilo de cada lado de un borde.


border-top-style. Estilo del borde superior.
border-right-style. Estilo del borde derecho.
border-bottom-style. Estilo del borde inferior.
border-left-style. Estilo del borde izquierdo.

Y por ultimo cambiar el color de cada lado de nuestro borde debemos modificar nuestra etiqueta y que al final termine en color. Al final el código quedaría de la siguiente manera:

Cambiar color de cada lado de un borde.


border-top-color. Color del borde superior.
border-right-color. Color del borde derecho.
border-bottom-color. Color del borde inferior.
border-left-color. Color del borde izquierdo.

Dato importante.


Si queremos modificar nuestro borde de igual manera para ambos lados en cuanto al color, estilo o ancho usamos las propiedades explicadas anteriormente pero sin especificar algún lado. border-width para el ancho, border-style para el estilo y border-color para el color.

Consejo.


Por preferencia cuando vamos a cambiar cada borde por separado debe ir en orden a las manecillas del reloj es decir, debe ir primero top, right, bottom y luego left para evitar errores de carácter en nuestro HTML y CSS, siempre los lados se colocan en sentido del reloj, de derecha a izquierda.

Resumir código en CSS.


Resumir nuestro código CSS es muy útil, ademas de evitar colocar demasiadas propiedades nos ayuda a evitar tener que escribir constantemente border-top, border-right.... etc, simplemente lo colocamos de la siguiente manera:

border-width: 5px 7px 5px 7px; De esta forma colocamos el ancho resumido en pixeles (solo cambiamos el valor numérico), se coloca en orden top, right, bottom, left. Sin separarlos por coma. (Se ordena de acorde a las manecillas del reloj).

border-style: groove dotted double dashed; De igual manera se coloca de derecha a izquierda, colocamos el nombre del estilo que tendrá cada lado, primero el superior, luego derecho, inferior y izquierdo, esos son los estilos que va a contener cada lado.

border-color: red yellow blue green; Colocamos el color que llevara cada lado sin separarlos por coma, primero se añade el color superior, derecho, inferior y izquierdo.

A continuación un ejemplo de un borde en CSS sin resumir y otro resumido. Como se puede apreciar en el ejemplo es mucho lo que evitamos escribir y podemos programar mas rápido. 


#border {
background-color: #000000;
font-family: verdana;
color: #ffffff;

border-top-width: 5px;
border-right-width: 7px;
border-bottom-width: 5px;
border-left-width: 7px;

border-top-style: groove;
border-right-style: dotted;
border-bottom-style: double;
border-left-style: dashed;

border-top-color: red;
border-right-color: yellow;
border-bottom-color: blue;
border-left-color: green;
}
Código resumido en CSS: 
#border {
background-color: #000000;
font-family: verdana;
color: #ffffff;

border-width: 5px 7px 5px 7px;
border-style: groove dotted double dashed; 
border-color: red yellow blue green;
}
