Diferentes propiedades para un elemento HTML.


CSS es un lenguaje de programación que nos permite definir mas de un atributo a un solo elemento HTML, gracias a esto podemos personalizar diferentes etiquetas y añadirles diferentes estilos, ademas de seleccionar una en especifico y añadirle estilos de forma individual o con otro conjunto de elementos.

En este ejemplo usaremos etiquetas para títulos (h1, h2, h3....), todos van a contener la misma tipografía y color de fuente pero la etiqueta <h1> va a contener un fondo (background-color) de color azul mientras los otros no contienen ningún fondo.


Así es como se visualizara nuestro efecto al final. Esto lo hacemos de la siguiente manera, abrimos nuestra etiqueta <style>; </style> (dentro de <head>), añadiremos nuestras propiedades de titulo y añadimos estilos, font-family para el estilo de texto y la propiedad color para el color de cada titulo. Luego podemos añadir un <h1> por separado con un color de fondo y un tamaño de texto diferente.

Con este ejemplo damos a explicar que podemos añadir a varias etiquetas HTML diferentes efectos y luego a una de ellas en especifico añadirle un efecto diferente a todas las demás.

Al final nuestro código debe quedar de la siguiente manera. 

<html>
<head>

<style>
h1, h2, h3, h4, h5, h6 {
  font-family: Verdana; color: red;
}

h1 {
  font-size: 40px; background-color: blue;
}
</style>

</head>
<body>

<h1>Este titulo tiene un color de fondo azul</h1>
<h2>Este titulo no tiene color de fondo</h2>
<h3>Este titulo no tiene color de fondo</h3>
<h4>Este titulo no tiene color de fondo</h4>
<h5>Este titulo no tiene color de fondo</h5>
<h6>Este titulo no tiene color de fondo</h6>

</body>
</html>
