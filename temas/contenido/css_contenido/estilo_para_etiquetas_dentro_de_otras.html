Estilo para etiquetas dentro de otras. Secuencia de etiquetas


Una de las funciones muy útiles de CSS es añadir estilo a una etiqueta dentro de otra, ejemplo. Podemos hacer que dentro de la etiqueta "<p>" la etiqueta "<i>" tenga otro color y estilo. Toda las propiedades de texto 'negrita' que estén dentro de la etiqueta párrafo <p> tendrán otro estilo (tamaño, color o tipo de fuente diferente).

A continuación un ejemplo de diferentes etiquetas que tienen estilos estando dentro de otras. Cualquier etiqueta dentro de otra puede tener un efecto único o diferente. 
<html>
<head>
<style>
h2 {color: green;}
h2 strong{color: red;}
p strong {color: blue;}
h3, strong {color: brown;}
</style>
</head>
<body>

<h2>Este ejemplo aparecerá de color verde, luego un elemento con la propiedad <strong>
strong</strong> dentro de un "h1" aparecerá rojo</h2>

<p>Este es otro ejemplo. Un párrafo que contenga la etiqueta <strong>
strong</strong> adentro aparecerá de color azul</p>

<h3>Este es otro ejemplo. La etiqueta <h3> y <strong> por si solas contendrán un
color marrón porque fueron separadas por coma; Esto indicas que la propiedad se aplica a 
ambas.</h3>

</body>
</html>

En el ejemplo anterior la sección de <head> etiqueta "style" hemos colocado las siguientes propiedades.

Todas las etiquetas <h2> serán de color verde.

Las etiquetas "strong" dentro de "<h2>" será de color rojo.

Strong (propiedad para texto en negrita) dentro de la etiqueta "<p>" aparecerá de color azul.

La propiedad <strong> y la propiedad <h3> serán de color marrón porque ambos fueron separadas por coma , y al ser separadas cada etiqueta va a contener el mismo estilo que le añadimos.
