Estilo para varios elementos con una sola propiedad CSS


Añadir varios estilos CSS a elementos HTML usando una sola propiedad es una de las funciones que nos permite CSS, esto es muy fácil, útil y complejo, no solo porque nos evita escribir más de la cuenta, si no porque ahorrar la escritura de etiquetas, nos evita añadir cada etiqueta una y otra vez para añadirle un determinado estilo; Y mientras menos código mucho mejor.

La sintaxis es disponer los nombres de los elementos HTML que queremos aplicar una regla separados por coma y luego añadirle los estilo que queremos. Ejemplo:

En este ejemplo usaremos la misma fuente y color para los elementos h1, h2 y h3. Los vamos a colocar dentro de <head> y luego dentro de <style>, cada uno separado por coma, Obvio los atributos no colocados adentro de nuestra propiedad "<style>" no cambiaran en absoluto. Veamos unas imágenes a continuación.

Así es como sera nuestro código: 
<html>
<head>

<style>
h1, h2, h3 {
  color:#fff; background-color: #000; font-family: Times New Roman;
}
</style>

</head>
<body>

<h1>Este atributo "h1" si cambio</h1>
<h2>Este atributo "h2" si cambio</h2>
<h3>Este atributo "h3" si cambio</h3>
<h4>Este atributo "h4" no cambio</h4>
<h5>Este atributo "h5" no cambio</h5>
<h6>Este atributo "h6" no cambio</h6>

</body>
</html>
