Formulario con estilo CSS.


Con HTML aprendimos añadir un formulario sencillo, ahora vamos aprender como añadirle estilos mediante CSS (color de fondo, tipo de letra, entre otras propiedades.) de una forma fácil y sencilla; Añadiremos las siguientes propiedades.

1. Añadimos un "div id o div class" a nuestro formulario en HTML y en CSS le añadiremos el div con el mismo nombre que le colocamos en HTML. Este indicara el ancho, alto, margen, padding, color de fondo, bordes, entre otras propiedades.

2. Añadimos nuestro div seguido de la palabra "label", antes se le debe anteponer el nombre del div colocado en HTML. Este se usara para modificar el tamaño de las letras, tipo de fuente, color del texto y posición de los label. Con esta propiedad indicamos los efectos que van a contener nuestra etiqueta label

3. Por ultimo añadimos la clase ".botonsubmit", que indica los efectos de nuestro botón. Con esta propiedad modificamos el estilo del botón "submit", para cambiar el color, color de fondo y margenes del botón, entre otros estilos.

Ejemplo de una estructura de un formulario con HTML y CSS. 
<html>
<head>
    <link href="estilo.css" rel="stylesheet" type="text/css"/> 
</head>
<body>

<div id="formulario">
<form method="post" action="#">
<label>Ingrese nombre:</label>
<input type="text" name="nombre" size="44">
<br><br>
<label>Ingrese mail:</label>
<input type="text" name="mail" size="44">
<br><br>
<label>Comentarios:</label>
<textarea name="comentarios" cols="30" rows="5"></textarea> 
<br><br>
<input class="botonsubmit" type="submit" value="Enviar/Confirmar">
</form>
</div>

</body>
</html>
Y en CSS ira de la siguiente manera. 
#formulario {
  width:500px; 
  margin:10px;
  background-color:#ffe;
  border:3px solid black;
  padding:10px;  
}
#formulario label {
  width:120px; 
  float:left;
  font-family:verdana;
  font-size:14px;
  color:#000000;  
}
.botonsubmit {
  color:yellow;
  background-color:#000000;
  border: 2px groove yellow;
}
