HTML con estilo CSS. <h1 style>


Podemos añadir estilos y propiedades de CSS en HTML de forma rápida y sencilla, aunque esta es una forma muy sencilla pero poco recomendable añadir estilo de esta forma. En este caso veremos un ejemplo con una etiqueta "H1, etiqueta para titulo" y mediante la propiedad style de CSS le daremos efectos de colores.


Como podemos observar, los dos estilos de nuestra propiedad para titulo (h1, h2, h3....) son exactamente iguales, pero estamos obligados a definir cada elemento con un estilo (sea igual o diferente el color).

Cada vez que iniciamos una propiedad (color, fondo, entre otros) debemos separarla de la siguiente por ; punto y coma.

Cuando definimos estilos directamente en HTML, tenemos que tener en cuenta que el estilo se aplica únicamente al elemento donde iniciamos la propiedad "style", es decir, si tenemos dos secciones "h1", deberemos definir la propiedad style para cada elemento.

Más adelante se explicaran muchas propiedades en CSS. En este primer ejemplo iniciamos las propiedades color (define el color del texto) y background-color (define el color de fondo del texto). 

<!DOCTYPE html>
<html>
<head>

<title>Empezando el curso de CSS en TU WEB Informativa</title>

</head>
<body>

<h1 style="color:white; background-color:black;">
Este mensaje es de color blanco sobre fondo negro. 
</h1>

<h3 style="color:#ff0000; background-color:#ffff00;">
Este mensaje es de color rojo sobre fondo amarillo. 
</h3>

</body>
</html>
