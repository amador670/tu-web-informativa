Margen de un elemento. (Margin CSS).


En el tema anterior se explico la propiedad padding de un elemento HTML, ahora explicaremos la propiedad margin que ademas de crear una especie de espacio entre el elemento y la pagina no se debe confundir con la propiedad padding, ya que ambas son diferentes.

La propiedad margin va después del padding y del borde, es el margen de cada elemento y puede ser añadido con la propiedad "margin", también se puede añadir a cada uno de sus lados por separado. El margin puede ser aplicado a todos los elementos HTML. A continuación una imagen donde se explica la propiedad padding, border, y margin; Resaltando el espacio que ocupa cada propiedad.


Como se explica en la imagen anterior podemos ver que el padding es el espacio entre el contenido y el borde y el margin es el espacio entre el borde y la pagina. A continuación un ejemplo de como se visualiza un texto con la propiedad margin.

Esta propiedad se añade muy fácil, un ejemplo sencillo y rápido es de la siguiente manera:

margin: 5px; (El numero es añadido en pixeles según la cantidad de espacio que necesitemos). O especificando cada uno de sus lados, ejemplo margin-top: 5px.
