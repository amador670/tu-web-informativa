Menú de forma horizontal. Menú mediante CSS.


Una de las mejores partes de CSS es elaborar nuestro propio menú y mas cuando lo crearemos por primera vez; Es muy sencillo crearlo, solo necesitamos de tiempo, paciencia y de cierto conocimiento. A continuación elaboraremos un menú, este puede ser modificado al gusto de cada usuario.

Para crear un menú con CSS debemos hacer una lista (Ejemplo para hacer una lista en HTML). Añadimos las propiedades "<ul>;</ul>", adentro de estas etiquetas deben ir "<li>;</li>, a continuación de estas etiquetas debe ir "<a href="nombre del archivo.html"></a>" para que al darle clic nos lleve a una categoría diferente, todo esto debe ir encerrado en un "<div>" para poder darle estilo. Nuestra estructura sera algo parecido al siguiente ejemplo.

<div>
<ul>
<li><a href="ruta del enlace">INICIO</a></li>
<li><a href="ruta del enlace">SERVICIOS</a></li>
<li><a href="ruta del enlace">CONTACTOS</a></li>
</ul>.
</div>.

A continuación vamos a ver un ejemplo de código de la estructura en HTML del menú que crearemos. 

<html>
<head>
    <link href="estilo.css" rel="stylesheet" type="text/css"/> 
</head>
<body>

<div id="menu">
  <!--Inicio del Menu-->  
<ul class="menu">

   <!--Elementos del Menu-->
 <li><a href=""> Inicio </a></li>
 <li><a href=""> Contenido </a><!--No cerrar "li"-->

   <!--Inicio de Sub-Menu-->  
<ul>
 <li><a href="">Sección I</a></li>
 <li><a href="">Sección II</a></li>
 <li><a href="">Sección III</a></li>
</ul>
   <!--Fin Sub-Menu-->

</li> <!--cerramos li-->

<li><a href="">Blog</a></li>
<li><a href="">Contactos</a></li>
</ul>
<!--fin del menu y cerramos nuestro div-->
</div> 

</body>
</html>

En el ejemplo anterior se observa un <div id> con el nombre de menú que contiene nuestra lista, luego tenemos un <ul class> con el nombre de menú y adentro tenemos nuestro <li> con unos <a href>, de esta forma estructuramos nuestro menú con HTML.

También podemos observar una parte que dice "Inicio de Sub-Menu" en esa parte podemos encontrar varias sub categorías, para crearlas solo tenemos que colocar otra etiqueta <ul> adentro de un <li> sin cerrar la etiqueta y adentro colocar el sub menú que queremos y al final cerrar nuestro </li>.

Bueno ya tenemos nuestro estructura en HTML que era lo mas fácil, en CSS vamos añadir todas las propiedades de efecto. Vamos a observar un ejemplo de CSS con las propiedades añadidas, el usuario puede probar con cada propiedad e ir probando con los diferentes efectos. 
/*Abrimos nuestra etiqueta <div> en CSS con el nombre que le colocamos en HTML en este caso "menú".*/

#menú {
 width: 500px; Este sera el ancho del menú (pueden modificar el valor).
 height: 40px; Este sera el alto del menú (pueden modificar el valor).
 margin: 0 auto; Este sera el margin del menú, en este caso sera 0  (pueden configurar el valor).
 background: silver; Este sera el color de fondo del menú (pueden modificar su color).
 border-radius: 5px; Este serán los bordes del menú (pueden quitarlo o modificar su valor).
}


/*Esta serán las propiedades para "menú" de nuestro "ul class" 
(el mismo nombre que le coloquemos en HTML debe ir aquí).*/

.menú{ 
 width: 400px; Este valor puede ir "0 auto" o podemos colocar el mismo 
 del "div #menú (500px)".
 height: 40px; Este sera el alto (pueden modificar el valor).
 margin: 10px; Este sera el espacio entre la pagina y el menú (pueden configurar el valor).
}

/*La etiqueta ".menú li" sera una configuración para nuestro menú etiquetas li.*/

.menú li{ 
 display: block; Con esta propiedad eliminamos los puntos que salen en nuestra lista.
 float: left; Es e orden de las letras de nuestro menú (en este caso de izquierda a derecha).
 height: 0 auto; Este sera el sombreado que aparece al pasar nuestro cursor por el menú 
 (Pueden modificar su valor pero "0 auto" es lo aconsejable).
 position: relative; Este sera la posición que tendrá nuestro menú.
}

/*La etiqueta "".menú li:hover" sera para añadir la transparencia de nuestro 
menú al pasar el cursor sobre el.*/

.menú li:hover{
 background: rgba(0, 0, 0, .3); En este caso el valor de transparencia es el 
 ultimo ".3" (Pueden modificar su valor).
}

/*Con esta propiedad podemos modificar los estilos que tendrá nuestro texto en 
cuanto al tamaño, alineación, fuente, entre otros.*/

.menú li a{
 display: block;
 text-align: center; Este sera el centrado de las palabras adentro del menús. 
 text-transform: uppercase; Esta sera la transformación de nuestro texto.
 text-decoration: none; Esta sera la decoración de nuestro texto, en este caso "ninguna".
 font-family: Arial; Este sera el estilo de texto de nuestro menú 
 (Pueden modificar su valor).
 font-size: 16px; Este sera el tamaño de las letras de nuestro menú 
 (Pueden modificar su tamaño).
 color: #000000; Este sera el color de las letras de nuestro menú, 
 (Pueden modificar su valor). 
 padding: 12px 23px; Este sera el padding entre nuestras letras y el menú. "12px" 
 para la altura y "23px" para el ancho(Pueden modificar su valor).
}

/*Estos serán los valores para la propiedad ".menú ul" que sera los valores de 
nuestros "ul" en HTML.*/
.menú ul {
 opacity: 0; La opacidad para nuestro menú.
 position: absolute; Posición para nuestro menú en cuanto a las etiquetas "ul".
 background: silver; Color para nuestro "sub-menú" en este caso nosotros elegimos el 
 mismo "silver" que coloque en el "#menú" (Pueden modificar su valor).
 border-radius: 5px; Este sera el redondeo de bordes para nuestras sub-categorías 
 (Pueden modificar su valor).
}

/*La propiedad ".menú li:hover ul " sera el opacidad que tendrá nuestro "sub-menú".*/

.menú li:hover ul {
 opacity: 1; En este caso el valor sera uno (1) (Pueden modificar su valor).
}

/*Esta propiedad ".menú ul li", indica los valores para nuestros "li" dentro de las 
etiquetas "ul".*/

.menú ul li {
 width: 200px; Este sera el ancho para nuestro sub-menú (Pueden modificar su valor).
 overflow: hidden; 
 border-top: 1px solid #000000; Este sera el borde superior de nuestro sub-menú 
 (Pueden modificar sus dos valores).
}

/*La propiedad ".menú li:hover ul li" sera parte de la configuración de nuestro sub-menú.*/

.menú li:hover ul li {
 height: 30px; Este sera el ancho de separación de nuestro sub-menú.
 overflow: visible;
}

/*Estas serán propiedades de nuestro sub-menú.*/

.menú ul li a {
 padding: 8px 0; Este sera el padding que tendrá el sub-menú.
 color: #000000; Este sera el color que tendrán las palabras del sub-menú 
 (Pueden modificar su valor).
}

/*Ultima propiedad para nuestro sub-menú.*/

.menú ul li:hover a {
 color:#ffffff; Esta propiedad indica el color del sub-menú al pasar el cursor. 
 (Pueden modificar este valor).
}
