Padding de un elemento HTML mediante CSS.


El padding de un elemento es una propiedad muy útil que CSS, nos permite añadir relleno o espacio a diferentes archivos (texto, imágenes, entre otros), esta propiedad crea una especie de relleno entre el contenido y el borde, permitiéndonos así que no se vea todo apiñado y sobrecargado.

Un padding puede ser aplicado a varios elementos; podemos configurar cada lado de padding a nuestro gusto (top, right, bottom o left) o simplemente podemos poner un valor estándar para todos los lados. A continuación un ejemplo de un texto que contiene la propiedad padding.


Como podemos ver en la imagen anterior se ha añadir un espacio, el texto esta encerrado entre un borde y se a separado de dicho borde. Un ejemplo de como añadimos la propiedad padding es de la siguiente manera:

padding: 5px. El numero debe ir colocado en pixeles.

También podemos configurar cada lado por separado, escribiendo "padding" y el lado que queramos cambiar en ingles. Ejemplo padding-top.
