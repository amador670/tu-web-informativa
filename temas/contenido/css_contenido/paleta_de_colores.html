Paleta de colores. Tipo de formato para colores


Formato de colores, con CSS podemos añadir colores a diferentes propiedades, como párrafos, títulos, entre otros, cada color puede ser colocado en ingles, es decir "red", "blue", "green", "yellow", etc. Puede ser colocado en carácter RGB y también en carácter hexadecimales, pero ustedes se harán la pregunta ¿Como saber que código poner para cada color?. Podemos entrar a paleta de colores donde encontraremos diferentes colores para elegir y usar a nuestro gusto, esta pagina generara nuestro código para cada color de forma hexadecimal y también de forma RGB.

Ejemplo de como añadir cada color:

#000000. Este color es negro. Cada ves que vamos añadir un color en caracter hexadecimal se antepone un "#" y después el código.

Red. Este color es rojo, se coloca el nombre del color en ingles.

rgb(255,255,255). Este color es blanco, de esta forma lo añadimos de forma RGB. (tiene un fondo negro para poder ver el color, en este caso blanco)
