Posicionamiento absoluto (position absolute).


Position: absolute;" propiedad para cambiar el espacio de un objeto; Al igual que position relative (tema anterior). Esta propiedad nos permite mover un archivo a diferentes posiciones de nuestra pagina. Las diferencias son que en position relative se conserva un margen al mover un archivo (imagen, párrafo, etc), mientras que en position absolute no se conserva.

Con esta propiedad de posicionamiento podemos desplazar el elemento a cualquier parte de la página, pero el espacio por defecto para dicho elemento queda vacío. A continuación una imagen de un ejemplo de "position: absolute;".


Como se puede ver en la imagen anterior esta no conserva ningún margen entre cada párrafo. Ejemplo de código HTML y CSS usando la propiedad position absolute.
<html>
<head>
    <link href="estilo.css" rel="stylesheet" type="text/css"/> 
</head>
<body>

<div id="caja1">
<p>Esta es la primer caja.</p>
<p>No se desplaza.</p>
</div>

<div id="caja2">
<p>Esta es la segunda caja.</p>
<p>Se desplaza con position absolute. Sus propiedades son left:40px; y top:210px</p>
</div>

<div id="caja3">
<p>Esta es la tercer caja.</p>
<p>No se desplaza.</p>
</div>

</body>
</html>
En este ejemplo añadimos nuestras propiedades en CSS de la siguiente manera. 
#caja1, #caja3 {
  background-color:#f99;
  font-family:verdana;
  font-size:1.3em;
}
#caja2 {
  background-color:#ff0;
  font-size:1.3em;
  position:absolute;
  left:40px;
  top:210px;
}
