Posicionamiento absoluto con dos columnas


Posicionar dos columnas como imágenes, párrafos, o cualquier otro elemento en dos columnas, usando la propiedad "position: absolute" de CSS. Este ejemplo es elaborado con la finalidad de que el usuario tenga un amplio conocimiento y elimine cualquier duda de la propiedad absolute además de brindar ayudar para aquellos que desean colocar 2 o mas columnas. Podríamos decir que este tema es extra y es con el fin de ayudar a nuestros usuarios con cualquier duda. Ejemplo del archivo CSS a continuación: 
* {
  margin:5px;
  padding:0;
  font-size:20px;
}
#columna1 {
  position:absolute;
  width:200px;
  background-color:#ffff55;
}
#columna2 { 
  margin-left:220px;
  background-color:#ffffbb;
}

Se añadieron dos <div> en HTML ("columna1" y "columna2") y en CSS hemos añadido un selector universal (no es obligatorio añadirlo) y a continuación hemos añadido en CSS los respectivos <div> con la propiedad "position: absolute" la diferencia entre ambos es que el div columna2 tiene la propiedad margin-left con 220px para que aparezca después que la columna1.

La columna1 tiene un width de 200px que es el ancho que ocupara la información que contenga ese <div>, luego la columna2 aparecerá 220px después de la columna1, esto significa que parecerá una al lado de la otra.
