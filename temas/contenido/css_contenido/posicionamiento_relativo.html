Posicionamiento relativo (position relative).


CSS nos permite configurar la posición de una imagen o texto en cualquier parte de nuestra pagina, en este ejemplo vamos a ver como posicionar una párrafo con la propiedad "position: relative" que nos brinda CSS. Con esta propiedad podemos hacer que un elemento en HTML se posicione de una manera diferente a la que aparece por defecto; Por lo general esta aparece configurada por el navegador como "position: static").

Con la propiedad "position: relative" configuramos un elemento de izquierda a derecha y de arriba abajo. Esta propiedad es muy útil cuando queremos posicionar un archivo en una parte en especifico. A continuación imágenes de un ejemplo:


Ejemplo en HTML y CSS de la imagen anterior de position: relative. 
<html>
<head>
    <link href="estilo.css" rel="stylesheet" type="text/css"/> 
</head>
<body>

<div id="caja1">
<p>Esta es la primer caja.</p>
<p>No se desplaza.</p>
</div>

<div id="caja2">
<p>Esta es la segunda caja.</p>
<p>Se desplaza 15 píxeles a la derecha y 5 hacia abajo de su posición 
por defecto.</p>
</div>

<div id="caja3">
<p>Esta es la tercer caja.</p>
<p>No se desplaza.</p>
</div>

</body>
</html>
En este ejemplo añadimos nuestras propiedades en CSS de la siguiente manera. 
#caja1, #caja2, #caja3 {
  background-color:#f99;
  font-family:verdana;
  font-size:1.3em;
}
#caja2 {
  position:relative;
  left:15px;
  top:5px;
}
