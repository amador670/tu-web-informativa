Propiedades relacionadas a fuente de texto


Mediante CSS podemos cambiar el color, tamaño, estilo entre otras características a nuestro texto. El día de hoy vamos aprender como podemos cambiar esto usando CSS adentro de un documento HTML. Veamos unos conceptos a continuación:

font-family. Con esta propiedad podemos cambiar el texto. En ocasiones cuando la fuente no este disponible el navegador selecciona la fuente que tiene por defecto. También podemos definir varias fuentes por si alguna no se encuentra disponible por el navegador, esto lo hacemos separando por coma.

Las propiedades para font-family mas comunes son:
  1. Arial.
  2. Arial Black.
  3. Arial Narrow.
  4. Courier New.
  5. Georgia.
  6. Impact.
  7. Tahoma.
  8. Times New Roman.
  9. Verdana.

Luego esta la propiedad font-size que indica el tamaño en pixeles (px) para nuestro texto o fuente.

font-style: Indica el estilo que tendrá nuestra fuente. Sus valores son normal, italic, oblique.

font-weight: Esta propiedad se encarga del peso de nuestra fuente es decir su relleno o contorno. Mientras mayor sea su atributo mas relleno tendrá cada palabra, texto o fuente.

Las propiedades que puede tener font-weight son:
  - normal  - bold  - bolder  - lighter  - 100  - 200  - 300  - 400  - 500  - 600  - 700  - 800  - 900

Por ultimo esta la propiedad font-variant. Este propiedad se encarga de mostrar las letras en mayusculas o minúsculas. "small-caps" o "normal".

Asi debe ir nuestro codigo: 
<!DOCTYPE html>
<html>
<head>

<style>
h1 {
     font-family: times new roman;
     font-size: 30px;
     font-style: italic;
     font-weight: bold;
}
h2 {
     font-family: comic sans ms;
     font-size: 25px;
     font-weight: bold;
     font-variant: normal;
}
p {
     font-family: verdana;
     font-size: 18px;
     font-style: italic;
}
</style>

</head>
<body>

<h1>Provando primer ejemplo</h1>

<h2>Provando segundo ejemplo</h2>

<p>Este es un párrafo de ejemplo. Este párrafo posee estilos mediante CSS</p>

</body>
</html>

Como podemos ver en nuestra etiqueta "<head>" hemos creado una etiqueta "style", adentro colocaremos todos nuestro estilos CSS, luego hemos indicado que a las propiedades h1 y h2 tendrán un estilo. Mediante los corchetes { ; } indicamos los estilos CSS (estos deben ir adentro) y cada propiedad debe finalizar con un punto y coma.

Después de aplicar los distintos cambios así es como se visualizara (en este caso nuestro ejemplo).
