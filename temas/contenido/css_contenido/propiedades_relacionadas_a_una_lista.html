Propiedades relacionadas a una lista.


Con HTML podemos añadir una lista de forma sencilla; Una lista ordenada o no ordenada y CSS nos brinda las propiedades necesarias para cambiar una lista sin usar HTML, de esta forma podemos crear diferentes listas y que todas tenga el mismo estilo. Podemos usar una lista para cualquier contenido que necesitemos que tenga un orden y con CSS podemos añadirle estilo. Las propiedades para configurar una lista son las siguientes.

list-style-type. Con esta propiedad podemos configurar el tipo de lista. Sus valores son.

1. none.
2. disc.
3. circle.
4. square.
5. decimal.
6. decimal-leading-zero.
7. lower-roman.
8. upper-roman.
9. lower-alpha.
10. upper-alpha.

list-style-position. Con esta propiedad podemos modificar la posición de nuestra lista. Sus valores son.

1. inside.
2. outside.

list-style-image. Con esta propiedad podemos añadir una imagen o una url a nuestra lista. Sus valores son.

1. none
2. url

Ejemplo de una lista mediante CSS


En la imagen anterior podemos observar una lista con efectos de estilos mediante CSS y ahora veremos como esta creada mediante CSS. Podemos probar cuantas veces queramos y colocar la lista de diferentes maneras o efectos. 
<html>
<head>
   <link href="estilo.css" rel="stylesheet" type="text/css"/> 
</head>
<body>

<ul class="vacio">
<li>Efecto vacio</li>
</ul>
<ul class="circulorelleno">
<li>Efecto circulo relleno</li>
</ul>
<ul class="circulovacio">
<li>Efecto circulo vacio</li>
</ul>
<ul class="cuadrado">
<li>Efecto cuadrado</li>
</ul>
<ul class="numerico">
<li>Efecto numerico</li>
</ul>
<ul class="romanominuscula">
<li>Efecto romano minuscula</li>
</ul>
<ul class="romanomayuscula">
<li>Efecto romano mayuscula</li>
</ul>
<ul class="letrasminusculas">
<li>Efecto letra minuscula</li>
</ul>
<ul class="letrasmayusculas">
<li>Efecto letra mayuscula</li>
</ul>

</body>
</html>
En CSS lo colocamos de la siguiente manera 
.vacio{
  list-style-type:none;
}
.circulorelleno{
  list-style-type:disc;
}
.circulovacio{
  list-style-type:circle;
}
.cuadrado{
  list-style-type:square;
}
.numerico{
  list-style-type:decimal;
}
.romanominuscula{
  list-style-type:lower-roman;
}
.romanomayuscula{
  list-style-type:upper-roman;
}
.letrasminusculas{
  list-style-type:lower-alpha;
}
.letrasmayusculas{
  list-style-type:upper-alpha;
}
