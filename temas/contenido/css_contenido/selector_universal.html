Selector universal en CSS.


Esta propiedad permite seleccionar todos los elementos de una página; Todas las propiedades que se encuentren dentro del selector estarán por defecto en nuestra página con el estilo que le apliquemos; Esto no evita que podamos cambiar los estilos de cada etiqueta por separado. Podemos añadir un selector universal con un * (asterisco) y adentro colocamos las propiedades que estarán por defecto en toda la pagina.

Habitualmente no se utiliza mucho un selector universal, ya que es difícil que un mismo estilo se le pueda aplicar a todos los elementos de una página. Nosotros consideramos que es útil si sabemos utilizar esta dicha propiedad... Veamos un ejemplo de un selector universal.

*{
margin:5px; padding:5px; color:blue;
}

En el ejemplo anterior se aplica un selector universal y todos los elementos tendrán un margin de 5px, un padding de 5px y todos los caracteres y letras tendrán un color azul. También podemos añadir un selector para diferentes etiquetas HTML y todas las etiquetas seleccionadas tendrán el efecto que coloquemos en el selector. Veamos unos ejemplos.

h1 { color: red; background-color:black;}

h6 { color: white; background-color: black; }

p {color: blue}

Todas las etiquetas seleccionadas tendrán por defecto un estilo seleccionado. Es decir todos los <h1> y <h6> que coloquemos por defecto tendrán un color y un background-color establecido y la etiqueta <p> (para los párrafos) tendrá un color de texto azul.

También podemos aplicar los mismos estilos a dos o más etiquetas diferentes, se pueden encadenar los selectores. Para ello, se incluyen todos los selectores separados por una , (coma) y las propiedades se aplicaran a las etiquetas que seleccionemos. A continuación un ejemplo.

h1, h3, h6 {
color: #8A8E27;
font-weight: normal;
font-family: verdana;
}

De esta manera nuestras etiquetas (h1, h3, h6) tendrán el mismo estilo. Otra función de un selector universal es que después de colocar las propiedades podemos modificarlas más adelante. Ejemplo, si queremos que en un "<h1>" dentro de un "<div>" tenga un color diferente lo podemos cambiar y este tendrá otro color o cualquier otro estilo.
