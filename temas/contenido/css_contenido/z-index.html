Superposición de objetos (z-index)


Una de las muchas propiedades que nos brinda CSS es el posicionamiento de un elemento; Esta propiedad permite configurar dicho elemento para que se superponga ante otro. z-index es una propiedad que permite darle prioridad a un elemento, es decir aparezca adelante que lo otros.

Cuanto colocamos varios elementos en el mismo espacio se superponen, y con la propiedad z-index determinamos cuales cubren a otros. El elemento con mayor valor en la propiedad z-index es el que superpone a los demás (es el que aparecerá de primero o sobre otro elemento). El valor de esta propiedad por defecto es auto y si lo modificamos puede ir desde 1 a infinito.

La propiedad z-index puede ser aplicada en cualquier ocasión que queramos superponer un elemento, esta propiedad es compatible con las propiedades de posicionamiento como "position:absolute", "position:relative", o "position:fixed".

A continuación un ejemplo de una imagen con tres <div> que poseen la propiedad z-index.


En HTML y de CSS colocaremos el código de la siguiente manera. 
<html>
<head>
    <link href="estilo.css" rel="stylesheet" type="text/css"/> 
</head>
<body>

<div id="caja1">
<p>Esta es la primer caja.</p>
</div>

<div id="caja2">
<p>Esta es la segunda caja. Que fue posicionada con z-index para que estuviera de primera</p>
</div>

<div id="caja3">
<p>Esta es la tercer caja.</p>
</div>
En CSS nuestro código lo vamos a colocar de la siguiente manera 
#caja1 {
  position:absolute;
  background-color:brown;
  right:960px;
  bottom:60px;
  width:200px;
  height:200px;
  z-index:1;
  font-size: 15px;
}
#caja2 {
  position:absolute;
  background-color:yellow;
  right:1000px;
  bottom:100px;
  width:200px;
  height:200px;
  z-index:3;
  font-size: 22px;
}
#caja3 {
  position:absolute;
  background-color:blue;
  right:980px;
  bottom:80px;
  width:200px;
  height:200px;
  z-index:2;
  font-size: 15px;
}
