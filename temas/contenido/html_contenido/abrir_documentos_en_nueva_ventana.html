Abrir enlace o documentos en una nueva ventana.

Evento <onclick>


Esta es una etiqueta de JavaScript empleada para eventos; Esta propiedad también puede ser empleada mediante HTML para abrir un enlace y que aparezca en otra pagina de nuestro navegador. Para esta opción usaremos la siguiente propiedad.

target="_blank" onclick="window.open(this.href, this.target, 'width=800,height=500'); return false;">

Con la propiedad anterior podemos abrir imágenes o enlaces en una nueva ventana, solo hay que añadir esa propiedad en nuestro "a href". En la propiedad width y height colocamos el numero de pixeles que va a contener nuestra ventana, si queremos que sea la pantalla completa escribimos "0auto". Un ejemplo completo seria:

<a href="nombre.html o enlace" target="_blank" onclick="window.open(this.href, this.target, 'width=800,height=500'); return false;">

Nosotros no recomendamos añadir este atributo ya que en ocasiones puede ser muy molesto para el usuario, por alguna razón existe la opción de los navegadores de "abrir enlace en una nueva ventana."
