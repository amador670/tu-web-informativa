Barra desplazadora (custom scrollbar)


La barra de desplazamiento o también llamada scrollbar es un objeto de la interfaz gráfica del usuario, permite mostrar una imagen o un texto en una sección o recuadro sin ocupar toda la pagina, permite ser mostrado en un espacio especifico y ser deslizados hacia abajo o arriba mediante un scroll. La barra de desplazamiento aparece por lo general a la derecha de la pantalla y la otra barra deslizante es de forma horizontal.

Una barra scrollbar es muy eficiente, permite que el usuario no tenga la necesidad de bajar en toda la pagina; En HTML es muy fácil colocar una barra scrollbar solo necesitamos el siguiente código. 
<html>
<head>
</head>
<body>

<div style="height:500px; width:500px; font-size:18px; color:#A6120D;
overflow:scroll; scrollbar-base-color:#52DC0D;"/>

Información que ira dentro del scroll

</div>

</body>
</html>

Como podemos ver en la etiqueta hay varios atributos. El significa cada uno de ellos es...

Width. Este permite configurar el ancho del scrollbar.

Height. Este permite modificar la altura que tendrá nuestra barra scrollbar.

Overflow:scroll. Propiedad obligatoria, esta indica que se mostrara una barra scrollbar

Font-size. Determinara el ancho de las letras.

Color. Determina el color de las letras dentro del scroll.

Scrollbar-base-color. Con ese atributo cambiamos el color a las barra de nuestro scrollbar. (Este atributo solo esta disponible en algunos navegadores modernos).
