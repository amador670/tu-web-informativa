Botones en HTML <input type button>


Agregar botones en HTML se realiza mediante la etiqueta <input type="buttom">, una forma divertida y muy útil es agregar botones con un <a href> añadirle un enlace y que el usuario al darle clic lo lleve a la opción que desean ir (en este caso el enlace), también podemos añadir botones y mediante JavaScript puedan cumplir alguna función dinámica. Un botón puede tener un sin fin de usos. A continuación un ejemplo sencillo de como añadir un botón.

<input type="button" name="nombre del botón" value="nombre que vera el usuario">

Con la etiqueta mostrada anteriormente podemos añadir un botón de forma fácil y sencilla.

Informacion extra:


Si queremos hacer un botón y al darle clic aparezca un mensaje debemos añadir la siguiente propiedad:

<input type="button" name="nombre del botón" value="nombre que vera el usuario" onclick='alert("este es el mensaje que aparecera al dar clic")'>
