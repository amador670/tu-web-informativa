Caja de comentarios. <input type>


Una caja de comentarios es algo sencillo de hacer y si tenemos conocimientos sobre PHP y base de datos mucho mejor ya que podemos enlazar nuestra caja de comentarios a una base de datos. En HTML vamos aprender hacer el cajón y hacer la estructura.

Para que los usuarios puedan enviar los campos ingresados y sean vistos por el administrador necesitamos emplear base de datos ya que sin ella solo estará el diseño de una caja de comentarios mediante HTML.

Etiqueta para la caja de comentarios: 
<html>
<head>
</head>
<body>

<form action="accion.php" method="post">

Ingrese su Nombre:
<input type="text" name="nombre" size="25">
<br>
<br>
Ingrese su Contraseña:
<input type="password" name="contraseña" size="21">
<br>
<br>
Añade un comentario
<br>
<textarea type="comment" name="comentarios" rows="5" cols="60"></textarea>
<br>
<input type="submit" value="enviar">
<input type="reset" value="borrar">

</form>

</body>
</html>

Explicación de cada etiqueta.


<form action="accion.php" method="post">. Esta propiedad la utilizamos para indicar hacia donde irán los datos al ser enviados al servidor, el servidor procesa los datos y envía una respuesta al usuario después de rellena el formulario y enviar los datos. En este caso se ejecuta la página accion.php (dependiendo del nombre que hemos añadido). En este fichero podríamos hacer que se muestre un mensaje como por ejemplo: "Hola amigo. Usted es bienvenido a nuestra pagina" después del usuario darle clic al botón "enviar"

Input type="..". Es usada para crear botones, barras para ingresar contraseñas, texto, checkbox, entre muchos tipos de datos. Esta propiedad especifica el tipo que sera el elemento. La propiedad que usaremos en este caso sera: Text, con esta propiedad especificamos que ingresaremos texto, esta propiedad es la que aparece por defecto.

Password. Con esta propiedad especificamos que ingresaremos una contraseña, lo cual indica que nuestros caracteres a ingresar serán codificados.

Comment. Esta propiedad indica que vamos a ingresar un comentario, lo cual permite colocar todo tipo de carácteres.

Submit. Indica la opción de enviar, con esta propiedad se suele colocar el botón "enviar".

Reset. Especificamos que al darle clic a esta opción se reinician los datos. Esta se usa para el famoso botón "borrar".

Name. Con esta propiedad indicamos el nombre que contiene. Este es el nombre que nos llega a la base de datos.

Size. Es el tamaño que va a contener nuestra barra de texto, ya sea para una contraseña o comentario, esta indicara el tamaño.

Value. Esta propiedad indica el valor inicial de nuestro control.

Rows. En este ejemplo fue usada para indicar el alto de nuestra caja de comentarios.

Cols. En este ejemplo fue usada para indicar el ancho de nuestra caja de comentarios
