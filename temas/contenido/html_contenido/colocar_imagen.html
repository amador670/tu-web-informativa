Colocar imagen o insertar una imagen en HTML.
<img src="imagen"/>


Colocar o insertar una imagen en HTML, es una de las cosas que más usaremos en una página web, de esta forma logramos que nuestra página tenga variedad de visualización tanto de imágenes como texto y le da un toque llamativo, evitando que el usuario se aburra de tanto texto.

Para añadir una imagen en HTML usaremos la etiqueta <img src="nombre de la imagen.jpg"/> y de forma extra le añadimos las siguientes propiedades.

style="width:"300px"; height:"300px";". Esto permite añadirle un tamaño a la imagen, la propiedad Style para indicar que la imagen tendrá un estilo, width es ancho y height es altura, también añadiremos la palabra “px” que significa pixeles o podemos colocarlo con un % (porcentaje) de ancho y una etiqueta opcional pero recomendada es "alt" para indicar una breve descripción de la imagen.

Ejemplo. 
<html>
<head>
</head>
<body>

<img src="imagen.jpg" alt="nombre que describe la imagen" style="width:300px; height:300px;"/>

</body>
</html>

Nota: La etiqueta "align" se usa para alinear cualquier tipo de archivo, en este ejemplo una imagen; Luego indicamos su posición. Left (izquierda), center (centrado), right (derecha). Estas propiedades han sido usadas como un ejemplo, no son necesarias para añadir una imagen
