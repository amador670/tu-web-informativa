DOCTYPE HTML, versión de HTML, programación web.


La propiedad DOCTYPE indica en que versión de HTML se ha programado la pagina, actualmente existen varias versiones (la mas actual es HTML5), podemos decir que programamos en HTML5 cuando usamos las etiquetas de esta versión. La organización que define los estándares para la web es W3C.

Utilizar en lo posible las directivas de este comité de estándares nos traerá como ventaja que nuestras páginas en un futuro sigan viéndose correctamente en las nuevas versiones de navegadores. La declaración del tipo de documento (DTD Document Type Declaration) se ubica en la primera línea del archivo HTML, es decir antes de la marca HTML.

El motor de visualización de HTML en los navegadores web modernos ejecutan el DOCTYPE, en donde el DOCTYPE en un documento es servido, esto determina el modo de visualización de cada pagina web.

La propiedad DOCTYPE es necesaria para decirle al navegador qué versión de HTML es la que se usa en la página. Si no se hace, el navegador puede procesar e interpretar la página de forma incorrecta.

La versión más actual de HTML es el HTML5 y propone que en la declaración del DOCTYPE solo indiquemos la cadena:
<!DOCTYPE html>

Las versión anterior de HTML es la 4.01. Para versiones antiguas usamos alguno de los siguientes doctype.

Declaración Transitional:
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">

El tipo de DOCTYPE Transitional permiten algunos elementos y atributos antiguos que están en desuso. Este incluye todos los elementos y cualidades de HTML 4 Strict pero agrega cualidades transitorias, elementos desaprobados o elementos obsoletos.

Declaración Strict:
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

EL HTML 4.01 Strict no permite marcado de presentación con el argumento de que debería usar CSS en su lugar para eso. Si declaramos este DTD el navegador pasará a actuar con el cumplimiento de los estándares. Este es el modo recomendado por el W3C, ya que es compatible con el CSS y puede ser interpretado correctamente por todos los navegadores.

Existen otras versiones de DOCTYPE, existen las versiones mas antiguas de XHTML 1.0 que pueden ser Transitional o Strict y las versiones de Frameset antiguas y las versiones 4.1.

Ejemplo de DOCTYPE en un documento .html 

<!DOCTYPE html>


<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" 
"http://www.w3.org/TR/html4/loose.dtd">


<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN"
"http://www.w3.org/TR/html4/strict.dtd">
