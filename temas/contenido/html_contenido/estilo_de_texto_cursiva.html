Texto en cursiva "<i>"


Texto de forma cursiva, ideal para añadir un toque animado a nuestra pagina y resaltar información importante (títulos o frases) para el usuario. Esto lo hacemos utilizando la etiqueta “<i>” o la etiqueta <em> que es otra forma validad de añadir este efecto.

Ejemplo: 
<html>
<head>
</head>
<body>

<i>Este texto contiene un efecto cursiva en HTML</i>

</body>
</html>
