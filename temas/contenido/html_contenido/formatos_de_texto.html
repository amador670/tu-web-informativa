Formatos de texto en HTML


Los formatos de texto, son etiquetas que permiten la modificación del texto y añadir diferentes formatos como negrita, cursiva, subrayado, tachado, entre otros los cuales permiten destacar palabras o frases en HTML. A continuación los diferentes formatos existentes:

Texto en negritaFormato en negrita. Se aplica con la etiqueta <b>; </b>
ItálicaFormato itálica. Se aplica con la etiqueta <i>; </i>
Negrita e itálicaFormato negrita e itálica. Se añade con <b><i>; </i></b>
SubrayadoFormato subrayado. Se utiliza la etiqueta <u>; </u>
EnfatizadoEste formato se añade con la etiqueta <em>; </em>
FuerteFormato fuerte. Se añade con la etiqueta <strong>; </strong>
Code textoFormato estilo codigo. Se añade con la etiqueta <code>; </code>
Citation textFormato para cita's. Se añade con la etiqueta <cite>; </cite>
Keyboard textEste formato se añade con la etiqueta <kbd>; </kbd>
Sample textEste formato se añade con la etiqueta <samp>; </samp>
Variable element textEste formato se añade con la etiqueta <var>; </var>
Texto grandeFormato de texto grande. Se añade con la etiquetas <big>; </big>
Texto pequeñoFormato de texto pequeño. Se añade con <small>; </small>
Superíndice (superior)Para añadir un formato con superíndice usamos <sup>; </sup>
Subindice (inferior)Para añadir un formato con subindice usamos <sub>; </sub>
TachadoFormato tachado. Se añade con las etiquetas <strike>; </strike>
