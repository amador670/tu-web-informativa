Formulario, cuadro de selección individual <select>


Formulario <select>, este nos permite seleccionar una opción entre varias, dependiendo de cómo lo configuramos podemos seleccionar una opción entre diferentes definiciones o varias opciones. A continuación las propiedades que usaremos en nuestro formulario.

 Name: Nombre del formulario select.

 Size: Establece la cantidad de opciones que a primera vista se verán en el formulario select.

 Option: Es la principal propiedad ya que permite acceder a cada opción del select.

 Disabled: Si su valor es "true", bloquea el select y no permite realizar ningún tipo de selección. Si su valor es "false"  contrarresta el efecto anterior.

 Multiple: Permite al usuario seleccionar más de una opción (mediante la tecla Ctrl + clic).

 Type: Informa si el select es del tipo múltiple o simple.

Explicada cada propiedad anterior podemos mostrar un formulario select y a continuación explicar como formarlo usando las distintas propiedad nombradas anteriormente. (A continuación una imagen de cómo se visualizara cada uno dependiendo de el atributo que usemos)


Como hacer un formulario dependiendo de las etiquetas seleccionadas. 

<html>
<head>
</head>
<body>



<select name="Que tan buena es la pagina">
<option value="1">Excelente</option>
<option value="2">Útil</option>
<option value="3">Mala</option>
</select>



<select name="Que tan buena es la pagina" size="3">
<option value="1">Excelente</option>
<option value="2">Útil</option>
<option value="3">Mala</option>
</select>



<select name="Que tan buena es la pagina" size="3" multiple="multiple">
<option value="1">Excelente</option>
<option value="2">Útil</option>
<option value="3">Mala</option>
</select>



<select name="Que tan buena es la pagina">

<optgroup label="Buenas opiniones">
<option value="1">Excelente</option>
<option value="2">Útil</option>
<option value="3">Muy buena</option>
</optgroup>

<optgroup label="Malas opiniones">
<option value="4">Mala</option>
<option value="5">Inútil</option>
</optgroup>

</select>

</body>
</html>
