Insertar documento HTML dentro de un documento HTML.

<iframe>


¿Que es un iframe?, son elemento HTML que permite insertar o incrustar un documento HTML dentro de un documento HTML principal. Gracias a ellos podemos agregar dentro de una pagina HTML otra que se visualizara en una parte de ella.

Con la etiqueta <iframe>, podemos insertar una página web dentro de otra, de la misma forma que insertamos un gráfico o una tabla, incluso un video de Youtube. El iframe es un objeto que se manipula con la misma libertad que cualquier otro elemento dentro del espacio HTML. En ocasiones un iframe se utiliza para mostrar contenidos externos como si fueran parte del mismo sitio. Anteriormente pocos navegadores soportaban el uso de iframe pero hoy en día muchos navegadores modernos soportado el uso de iframe, como Google Chromer, Firefox, Safari, Opera, entre otros.

Iframe para uso de publicidad y sitios de colaboración.


Con el iframe los banners cobran una nueva dimensión. El anunciante puede colocar dentro de un sitio una página web que va más allá del banner, ya que el usuario podrá ser capar de rellenar un cuestionario, consultar paginas sin abandonar el sitio del anunciante.

Iframe, etiqueta y atributos.


Etiqueta principal: <iframe src="NOMBRE DEL ARCHIVO.html"></iframe>

src="nombre del archivo o dirección url que aparecerá en el iframe"

width="950px". Determina el ancho del recuadro que tendrá nuestro iframe. En este caso se eligió "950px", el valor puede ser modificado.

height="550px" Determina el alto del recuadro que tendrá nuestro iframe. En este caso se eligió "550px", el valor puede ser modificado.

scrolling="auto-yes-no". Añadimos esta etiqueta si la pagina que contiene el iframe es muy grande, con esta podemos desplazarnos a través de ella con un scroll. Podemos añadir "no" y evitar que se coloque o "si" y este se visualizara y también puede ser colocado de forma "auto" dependiendo del tamaño.

style="border:1". Con este atributo determinamos el borde que aparece alrededor de nuestro recuadro del iframe.

style="border:none" Con este atributo eliminamos el border de nuestro cuadro de iframe.

Existen muchos otros atributos pero los nombrados anteriormente son los principales, además de que muchas etiquetas de los iframe han sido eliminadas y ya no son soportadas en HTML5.

Aunque el uso del iframe no es muy común no está demás saber sobre esta etiqueta y sus características. Nosotros consideramos que es una etiqueta muy útil y eficaz.

Ejemplo de una etiqueta iframe sencilla para colocar en un sitio web: 

<html>
<head>
</head>
<body>

<iframe src="documento.html o url" width="950px" height="550px" scrolling="auto" 
style="border:none"></iframe>.

</body>
</html>
