Span, div id y div class. Propiedades de Desarrollo Web


La propiedad span es una propiedad en HTML usada principalmente para el diseño, se suele utilizar para añadir estilos mediante CSS o en HTML para atribuir colores, fondos, entre otros. Es una etiqueta que por si sola no cumple ninguna función pero usándola con efectos visuales es muy útil. Al igual que las etiquetas div class="." y div id="." son usadas para añadir estilos pero estas son comúnmente usadas cuando programamos con CSS. La etiqueta <span> y las etiquetas <div> son contenedores, son neutro y no añaden nada al documento.

Supongamos que tenemos un párrafo y queremos añadirle un color y un fondo, podemos usar la propiedad span. Ejemplo: 
<html>
<head>
</head>
<body>

<span style="color:red; background:black;">
Este texto sera rojo con un fondo negro.
</span>

</body>
</html>

Con la etiqueta span podemos atribuir muchos efectos visuales e incluso añadir iconos en nuestra web. Si deseas saber mas propiedades de diseño puedes revisar nuestro curso gratis de CSS.
