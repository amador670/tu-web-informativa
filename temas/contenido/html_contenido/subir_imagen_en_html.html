Subir o cargar imagen mediante HTML


Con esta propiedad podemos subir o cargar imágenes a HTML, esto puede ser usado para que el usuario establezca una foto de perfil, cargue una foto a su cuenta, entre otras opciones, cabe recordar que para que esto sea posible tiene que estar enlazado a una bases de datos (para mas información revisar la sección PHP (Lenguaje de programación)).

A continuación vamos aprender hacer la estructura mediante las etiquetas de HTML, esto es sencillo de elaborar, solo usaremos la etiqueta <input type="file">. Veamos un ejemplo: 

<html>
<head>
</head>
<body>

Ingrese su nombre:
<input type="text" name="nombre" size="30">
<br>
Seleccione una foto:
<input type="file" name="foto">
<br>
<input type="submit" value="Enviar">

</body>
</html>
