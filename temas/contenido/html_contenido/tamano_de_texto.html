Tamaño de texto <h1>, <h2>, <h3>, <h4>, <h5>, <h6>.


Esta etiqueta es usada principalmente para añadir títulos, se usa para resaltar el encabezado de algún articulo. Para cambiar el tamaño a una palabra o texto en HTML se usan las etiquetas <h1>, <h2>, <h3>, <h4>, <h5>, <h6>.

El orden del tamaño de cada una es dependiendo del numero que usamos, el numero mayor es el mas pequeño y el numero menor es el mas grande.

Ejemplo: 
<html>
<head>
</head>
<body>

<h6>Esta etiqueta posee un tamaño muy pequeño<h6>
<h5>Esta etiqueta posee un tamaño pequeño<h5>
<h4>Esta etiqueta posee un tamaño intermedio pequeño<h4>
<h3>Esta etiqueta posee un tamaño intermedio grande<h3>
<h2>Esta etiqueta posee un tamaño grande.<h2>
<h1>Esta etiqueta posee un tamaño muy grande.<h1>

</body>
</html>

Así se deberían visualizar en nuestra pagina web.
