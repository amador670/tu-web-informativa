Titulo para una página web. <title>


Para añadir un titulo o nombre para nuestra web usamos la etiqueta <title>; </title> esta etiqueta no se coloca en la sección “body” que es el cuerpo de la pagina, se coloca en la cabecera que es el “head” ahí es donde colocamos nuestra etiqueta. La mejor opción es poner un nombre corto y preciso y no uno demasiado exagerado. Si queremos evitar problemas con el titulo lo mejor es no colocar caracteres o símbolos, solo texto. A continuación un ejemplo.

Este titulo es el que aparecerá en la parte superior de nuestra pagina, es el titulo ubicado en las pestañas del navegador. 
<html>
<head>

<title> Titulo que va a contener la pagina web. Este sera el titulo que aparecerá
en la cabecera de la pestaña </title>

</head>
<body>
</body>
</html>
