Curso de CSS3 (Desarrollo Web)


En este curso aprenderemos como CSS3 afecta a HTML5 y como implementarlo, curso ideal si ya dominas o tienes conocimientos basicos en HTML.

¿Qué es CSS3?


Su nombre en español significa Hoja de Estilo Encascada y en ingles es Cascading Style Sheets como sus siglas lo indican.

Este lenguaje esta constituido por etiquetas simples, CSS3 es la ultima versión es usado para darle estilo a HTML, es usado en un documento aparte y es enlazado a HTML o también puede ser colocado dentro de HTML, lo cual es poco recomendable y no permite una modificación amplia para darle estilos. CSS como CSS3 es soportado en la gran mayoría de navegadores. Una de las principales características de CSS3 es su flexibilidad y las diferentes opciones que ofrece para realizar una misma tarea.

Una pagina web sin CSS en su interfaz es una pagina simple, solo se visualizaría la información. CSS es usado para definir estilos en un documento creado con HTML, actualmente existe una entidad llamada World Wide Web Consortium (W3C) que es el encargado de formular la especificación de las hojas de estilo que servirán de estándar para los agentes de usuario o navegadores.

¿Cómo se enlaza CSS y HTML?, ¿Cuál es la relación entre ambos lenguajes?


Es muy fácil enlazar CSS con HTML, esto se logran en la etiqueta <head></head> o mejor conocida como cabecera de HTML que se enlazan los contenidos CSS, dentro de este se añade la etiqueta "<link href="nombre del archivo.css" rel="stylesheet"/>", con esta etiqueta podemos enlazar diferentes archivos de extensión .css.

El significado de cada atributo de la etiqueta de CSS es:


   rel: Indica el tipo de relación que existe entre el recurso enlazado (archivo CSS) y la página HTML. Para los archivos CSS, siempre se utiliza el valor stylesheet.

   type: Indica el tipo de recurso enlazado. Sus valores están estandarizados y para los archivos CSS su valor siempre es text/css.

   href: Indica la dirección o url del archivo CSS que contiene los estilos. La dirección puede ser de un archivo interno o externo como en este caso una url.

La relación entre CSS y HTML es mucha, ya que con CSS podemos añadir una gran variedad de estilos, sin este nuestra pagina se vería simple, seria solo texto. Todas las paginas de hoy en día por mas pequeñas que sean poseen CSS en su estructura.

Compatibilidad con CSS y distintos navegadores.


Hoy en día la ultima versión CSS3 tiene compatibilidad con la mayoría de los navegadores, es decir que así como lo configuramos en Firefox debería visualizarse en Google Chromer, aun que cabe destacar que hay ciertos navegadores que no soportan bien algunos atributos o se visualizan diferente, por ejemplo solo la versión de Internet Explorer 10.0 soporta todos los selectores y demás propiedades de CSS3 mientras que en antiguas versiones no soporta algunas propiedades. Navegadores como Firefox, Chrome, Safari y Opera son mas avanzados en cuanto al soporte de CSS3.

¿Cuál es la extensión para un archivo CSS?


Un archivo de tipo CSS no es más que un archivo de texto cuya extensión es .css; Se pueden crear todos los archivos CSS que sean necesarios y cada página HTML puede enlazar una gran variedad de archivos CSS como lo necesite. Incluso podríamos crear un archivo CSS con un solo estilo para cada etiqueta y aun así HTML lo soportaría sin ningún problema, claro esto no es lo mas recomendado ya que trabajaríamos mas de la cuenta.
