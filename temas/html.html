Curso de HTML5 (Desarrollo Web)


Este curso va dirigido para los usuarios que desean aprender sobre HTML, su uso, etiquetas, como usarlo y lo mas importante como hacer una página web. En este curso hay contenido sobre etiquetas básicas y avanzadas de HTML5 y como crear una página web de una manera correcta.

El aprendizaje dependerá de cuánto interés tenga el usuario en aprender mediante la práctica y la constancia. No olvidemos que la práctica diaria es muy buena para no ir olvidando temas y etiquetas aprendidas...

¿Qué es HTML5?.


Su nombre en español es Lenguaje de Marcado de Hipertexto y en ingles es HyperText Markup Language como sus siglas lo indican HTML en esta version la ultima hasta la fecha que es HTML5.

Este lenguaje está constituido de elementos que el navegador interpreta y despliega en la pantalla de acuerdo a su objetivo. Con HTML podemos disponer imágenes sobre una página, listas, tablas para tabular datos, entre otros, mediante etiquetas o códigos que el navegador reconoce e interpreta.

En pocas palabras; Todas las páginas web están constituidas por etiquetas de HTML, desde Google, Facebook, Twitter, Youtube entre muchas otras como Blogger, solo que estas páginas poseen otros lenguaje de programación.

Ya sabemos que es HTML, ahora podemos ir adentrándonos un poco más en su practica que es mediante etiquetas. Empecemos por conocer más de ellas...

El lenguaje HTML está constituido por etiquetas.


El inicio de cada etiqueta se abre con “<>” y el fin para cada etiqueta es "</>", para cerrar una etiqueta se usa un “/”. Ejemplo digamos que queremos poner un titulo a una página, se usa la siguiente propiedad "<title> Aprendiendo HTML </title>”, como podemos apreciar lleva una etiqueta de comienzo y una de cierre además de una propiedad de HTML que es "title". Todas las etiquetas están en ingles así que debemos tener un mínimo conocimiento para facilitar el aprendizaje aunque no es obligatorio pero si ayudara mucho.

¿Cómo puedo crear una página HTML?.


Para crear una página web es muy fácil, se puede crear desde un simple editor block de notas hasta en Microsoft Word aunque nosotros recomendamos editores de texto para HTML que facilitan la creación para una página web. Existen programas como Notepad++, Sublimet Text 3, Brackets o Visual Studio Code, el cual puede usar el de su preferencia y comodida.

Estructura para crear una página web HTML.


Para crear una página web siempre vamos a usar las mismas etiquetas principales, la base de todo. Usaremos una etiqueta con la cual abriremos nuestra base que será <html>, luego añadimos <head> donde se introduce el titulo y opciones para el buscador y cerramos nuestra etiqueta </head>, luego usaremos la etiqueta <body> que sera el cuerpo de la página donde introduciremos nuestras etiquetas y todo lo que se visualizara en nuestra página y cerramos la etiqueta </body> y al final cerramos la etiqueta </html>. Extensión .html (guardar archivo)


Ya sabemos que para crear un documento HTML se emplea con un editor de texto o block de notas pero ahora debemos saber que al haber creado nuestra estructura en HTML debemos añadir al final del guardado .html lo cual permitirá que sea interpretado como una página web y no como un archivo de texto.
