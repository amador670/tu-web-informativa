Curso de JavaScript (Lenguaje de programación)


Este es un lenguaje de programación orientado a objetos, es un lenguaje de programación dinámico que se interpreta del lado del usuario a diferencia de PHP (Lenguaje de programación) que es orientado del lado del servidor. Este curso gratuito esta adaptado para todo tipo de usuario y es explicado de la mejor forma posible para su comprensión.

¿Que es JavaScript?


Es un lenguaje de programación orientado en objetos que es implementado del lado del usuario (client-side), es un lenguaje dinámico que permite la realización de efectos de forma dinámica, como efectos a menús, avisos, efectos al hacer clic o hacer un scroll entre muchas otras funciones, implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y de las páginas web. JavaScript posee ciertas similitudes y una sintaxis similar al lenguaje de programación C.

JavaScript es una marca registrada de Oracle Corporation, es usada con licencia por los productos creados por Netscape Communications y entidades actuales como la Fundación Mozilla.10 11

¿Como implementar JavaScript en un navegador?


Antiguamente JavaScript era implementado en pocos navegadores, estos no soportaban su semántica pero en la actualidad la mayoría de navegadores modernos (Google Chromer, Firefox, Opera e incluso las ultimas versiones de Internet Explorer) soportan este lenguaje de programación; Aun así todos ellos cuentan con la función de desactivar JavaScript lo cual evita mostrar los efectos e implementar este lenguaje.

Tradicionalmente se venía utilizando en páginas web HTML para realizar operaciones y únicamente en el marco de la aplicación cliente, sin acceso a funciones del servidor. Actualmente es ampliamente utilizado para enviar y recibir información del servidor junto con ayuda de otras tecnologías.

¿Como añadir JavaScript a mi pagina HTML?


Podemos emplear nuestro documento Javascript de una manera muy fácil en nuestra pagina web HTML, simplemente lo colocamos en el "body" (cuerpo de la pagina) en la linea que queramos que aparezca, para eso usamos la siguiente etiqueta

<script type="text/javascript">
Código JavaScript que usaremos
</script>

De esta forma podemos emplearlo en nuestro archivo .HTML pero la mejor forma es crear un documento a parte de nuestro HTML y añadirle la extensión .js al final, de esta forma indicamos que sera un archivo JavaScript; Luego de crear nuestro archivo .js añadimos en el "head" (cabecera de nuestra pagina web) la siguiente etiqueta

<script type="text/javascript" src="nombre del archivo.js"></script>

Errores al programar con JavaScript


A diferencia de otros lenguajes de programación, JavaScript es un lenguaje de programación interpretado, lo que significa que no se pueden detectar la mayoría de errores en el código hasta que se ejecutan, de forma que antes de considerar que se ha escrito bien un script es necesario probarlo en todos los navegadores que se vaya a utilizar para descartar errores

Cuando se producen errores durante la ejecución de un script, los navegadores proporcionan cierta información útil para descubrir el punto exacto en donde se ha producido el error y su posible solución. Desafortunadamente, no todos los navegadores proporcionan la misma información útil, lo que complica la solución de los errores para cada tipo de navegador.
