Curso de PHP (Lenguaje de programación)


Curso gratuito de PHP Es un lenguaje de programación fácil de aprender y mas si conocemos otros lenguajes de programación. Cabe destacar que cada usuario debe implementar cada ejemplo por su parte para un mayor aprendizaje.

¿Qué es PHP?


Sus siglas son un acrónimo de Hypertext Preprocessor, un lenguaje de código abierto muy popular especialmente adecuado para el desarrollo web, usado para páginas dinámicas y programación del lado del servidor. Puede ser incrustado en HTML, es un lenguaje open source, es decir que permite introducir modificaciones.

PHP es un lenguaje multiplataforma, esto permite que pueda ser soportado en distintos sistemas operativos, desde Windows, Mac, Linux entre otros; También ofrece soporte a otras bases de datos más como SQL Server, MySQL, PosgresSQL, Oracle, etc.

PHP es muy fácil de aprender por su simplicidad para el principiante, pero a su vez ofrece muchas características avanzadas para los programadores profesionales.

¿Diferencias entre PHP y otros lenguajes de programación?


Una de las grandes diferencias entre PHP y muchos otros lenguajes de programación es que este se ejecuta en un servidor, es uno de los primeros lenguajes basados en la programación orientada en un servidor además de ser uno de los más flexibles y potentes lenguajes.

En PHP el código es interpretado por un servidor web con un módulo de procesador de PHP que genera la página web resultante.

¿Qué podemos hacer con PHP?


PHP nos brinda la creación de páginas dinámicas. Son aquellas que piden información al servidor para actualizarse o cambiar sus datos, como ejemplo una paginas de noticias. Con HTML podemos crear formularios, tablas, encuestas o formularios para comentar y con PHP enviar la información a un servidor donde podemos ver y revisar la información que ha emitido el usuario.

El servidor web puede ser configurado incluso para que procese todos los ficheros HTML con PHP, por lo que no hay manera de que los usuarios puedan saber qué se está ejecutando. Actualmente el PHP suele incrustarse dentro del código HTML de las páginas web y ejecutarse desde un servidor.

Las páginas generadas con PHP necesitan ser cargadas por un servidor web para que sean procesadas y luego enviadas al navegador del usuario. Si no tenemos un servidor activo no se visualizaran las funciones de PHP o podrían dar error.

¿Cómo implementar PHP?


PHP se suele implementar junto a HTML en un mismo documento, pero este se separa completamente del código. Un ejemplo de cómo implementar PHP sería el siguiente.

<html>
   <head>
       <title>Ejemplo de PHP en TU WEB Informativa</title>
   </head>
   <body>
       <?php
       echo "¡Hola mundo, este es un ejemplo de PHP ~ curso gratuito!";
       ?>
   </body>
   </html>

El código de PHP está encerrado entre las etiquetas de comienzo y final <?php y ?> que permiten entrar y salir del "modo PHP". El propósito de estas etiquetas es separar el código PHP del resto de código, como por ejemplo el HTML.

PHP posee variables que se prefijan con el símbolo del dólar ($) y no es necesario indicar su tipo. Las variables, a diferencia de las funciones, distinguen entre mayúsculas y minúsculas.

Los caracteres pueden ser encerrados en dobles comillas como en comillas simples.

Los comentarios se pueden escribir bien con dos barras al principio de la línea // o de forma de comentarios multi-línea con /* */.

¿Cómo implementar PHP en un servidor?


Para dicho proceso tendríamos que comprar un servidor en línea y mantener nuestra página alojada en dicho servidor y dominio, pero existen muchos servidores gratuitos que permiten implementar nuestros archivos PHP desde nuestro sistema operativo.

Servidores de forma local, son aquellos que podemos usar para trabajar en nuestro proyecto de forma offline:

XAMPP un servidor independiente de plataforma y software libre, que consiste principalmente en la base de datos MySQL y servidor Web, y WAMP Server un servidor ligero y muy útil que nos proporciona muchas funciones al igual que XAMPP.
